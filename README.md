# Expense Tracker

A Django-based expense tracker web application for managing expenses, budgets, and shared spending.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run migrations:
   `python manage.py migrate`
4. Start the server:
   `python manage.py runserver`

## Project Structure

- `expense_tracker/` Django project and app files
- `requirements.txt` Python dependencies
